## Nivel 1: System Context Diagram

Quién usa el sistema y con qué aplicaciones externas se comunica.

```mermaid
C4Context
    title System Context — Todo List

    Person(user, "Usuario", "Persona que gestiona sus tareas desde la terminal")

    System(todoApp, "Todo List App", "Aplicación de consola para crear, editar y organizar tareas personales")

    System_Ext(notifService, "Servicio de Notificaciones", "Envía recordatorios por correo o push cuando una tarea está próxima a vencer")
    System_Ext(calendarApp, "Aplicación de Calendario", "Sincroniza tareas con fechas límite en el calendario del usuario")

    Rel(user, todoApp, "Crea, edita, lista y duplica tareas")
    Rel(todoApp, notifService, "Envía eventos de tareas pendientes", "HTTP/JSON")
    Rel(todoApp, calendarApp, "Sincroniza tareas con fecha límite", "HTTP/JSON")
```

Qué hay dentro del "Backend": UseCases, Commands, Queries y Repositories.

```mermaid
C4Component

    Rel(dupTask, dupTaskCmd, "Ejecuta")

    Rel(createAccount, createAccCmd, "Ejecuta")
    Rel(login, loginQry, "Ejecuta")

    Rel(createTaskCmd, iTaskRepo, "Usa")
    Rel(editTaskCmd, iTaskRepo, "Usa")
    Rel(dupTaskCmd, iTaskRepo, "Usa")
    Rel(listQry, iTaskRepo, "Usa")
    Rel(createAccCmd, iUserRepo, "Usa")
    Rel(loginQry, iUserRepo, "Usa")

    Rel(iTaskRepo, taskRepo, "Implementado por")
    Rel(iUserRepo, userRepo, "Implementado por")
```

## Nivel 4: Code Diagram

Detalle de `DuplicateTaskUseCase` — cómo se conectan las clases a nivel de código.

```mermaid
classDiagram
    class IUseCase~TIn_TOut~ {
        <<interface>>
        +ExecuteAsync(request TIn, ct CancellationToken) Task~TOut~
    }

    class DuplicateTaskUseCase {
        -_command ICommand~DuplicateTaskRequest_DuplicateTaskResult~
        +DuplicateTaskUseCase(command)
        +ExecuteAsync(request, ct) Task~DuplicateTaskResult~
    }

    class ICommand~TRequest_TResult~ {
        <<interface>>
        +ExecuteAsync(request TRequest, ct CancellationToken) Task~TResult~
    }

    class DuplicateTaskCommand {
        -_taskRepository ITaskRepository
        +DuplicateTaskCommand(taskRepository)
        +ExecuteAsync(request, ct) Task~DuplicateTaskResult~
    }

    class ITaskRepository {
        <<interface>>
        +GetByIdAsync(id, ct) Task~TaskItem~
        +GetByUserIdAsync(userId, ct) Task~IEnumerable~TaskItem~~
        +AddAsync(task, ct) Task
        +UpdateAsync(task, ct) Task
    }

    class DuplicateTaskRequest {
        +TaskId Guid
        +UserId Guid
    }

    class DuplicateTaskResult {
        +Task TaskItem
        +Error string
    }

    IUseCase <|.. DuplicateTaskUseCase : implements
    DuplicateTaskUseCase --> ICommand : uses
    ICommand <|.. DuplicateTaskCommand : implements
    DuplicateTaskCommand --> ITaskRepository : uses
    DuplicateTaskUseCase ..> DuplicateTaskRequest : input
    DuplicateTaskUseCase ..> DuplicateTaskResult : output
```
